Hypervis
==========
A Hypergraph visualisation tool written in Processing language. It supports subset-based, edge-based and Zykov drawing of hyperedges.

How to use
==========
* ``Processing 3:`` install it at https://processing.org/
* Open ``Hypervis.pde`` and run
* ``Demo video:`` https://www.youtube.com/watch?v=16iXlXGsUf4
